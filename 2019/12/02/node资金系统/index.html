实现的功能 和接口

注册

/api/users/register

方法： POST

根据注册使用的邮箱查找数据，若无则成功注册。若有，返回”该邮箱已被注册”提示。注册成功之后保存到mongodb数据库中。

  • 引入body-parser中间件获取POST传输数据

  • 密码加密保存：

    使用 bcrypt 函数 genSalt() 。

    语法：

  • mongodb 数据操作API

    • findOne 只返回一个文档。相比较于find 是返回全部符合条件的文档

  • 保存一条数据

登录

/api/users/login

方法：POST

条件1：邮箱 条件2：密码。先核对邮箱，是否存在，不存在即：“输入账号不存在”,存在再验证密码。密码验证无误之后，为该用户添加token令牌。

新知识点：

1.使用jwt

jwt.sign("规则","加密名字","过期时间","箭头函数");

  • 加密的密码验证

bcrypt.compare(用户输入的密码，数据库中的密码)

  • 为字段添加token

passport： 验证token

使用passport-jwt验证token

passport-jwt是一个针对jsonwebtoken的插件，passport是express框架的一个针对密码的中间件

1.安装

npm install passport-jwt -D

npm install passport

2.导入

在入口文件导入：
const passport = require("passport")

入口文件中初始化：
app.use(passport.initialize());

3.配置

config文件夹下新建 passport.js文件。并在入口文件中导入

require("./config/passport")(passport);
