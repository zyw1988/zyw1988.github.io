MUI

官网

优点： 有底部tabbar、常用icon。

缺点：是要下载复制到对应文件夹使用的，css样式和js需要手动导入。

mint-ui

官网

优点：偏 移动端，譬如Toast 弹窗提示使用居多，messageBox提示框、选项卡、顶部Header、底部Tabbar、导航栏navbar

element-ui

官网

vux

一个凑合的 Vue.js 移动端 UI 组件库

目前所知是 不能在vue-cli中使用的

swiper 移动端触摸滑动组件

使用

npm i --save swiper

使用必须条件：

内容物由 content 包裹，且长度/宽度 必须大于最外层wrapper。
